Synopsis
```
cp [OPTION]... [-T] SOURCE DEST
cp [OPTION]... SOURCE... DIRECTORY
cp [OPTION]... -t DIRECTORY SOURCE...
```

options
* -a, --archive : same as -dR --preserve=all
* -f, --force : if 
* -R, -r, --recrsive : recursive
* -s, --symbolic-link : make symbolic links instead of copying
* -v, --verbose
* --help : display help
